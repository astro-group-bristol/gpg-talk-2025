# GPG, encryption, and digital signatures

An [Astrophysics Developer Group](https://github.com/astro-group-bristol/developer-group) talk.

To build the slides, use [Typst](https://typst.app)

```bash
typst compile presentation/main.typ
```

Requires the following fonts:
- [Atkinson Hyperlegible Next](https://www.brailleinstitute.org/freefont/)
- [Adelphe Germinal](https://typotheque.genderfluid.space/fr/fontes/adelphe)
- [Alma Virgo](https://fontesk.com/alma-virgo-font/)
